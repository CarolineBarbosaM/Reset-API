# BACK-END API RESET NODE.JS

## Executando o Projeto

### 1 - Clone o reposotório 

```bash
git clone https://github.com/CarolineBarbosaM/Reset-API.git
```

### 2 - Instale as Dependências: 

```bash
npm install
```

### 3 - Configurações de acesso ao Banco de Dados no .env
```
############## DATABASE ################

CONNECTION=mysql
HOST=localhost
PORT=3306
USER=root
PASSWORD=11111111
DATABASE=teste
```

### 4 - Execute o projeto

```bash
npm start
```

### 5 - Acesse o projeto

```bash
localhost:3000
```
